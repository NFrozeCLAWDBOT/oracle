# Oracle

Interactive LLM Security Assessment Tool mapped to the OWASP Top 10 for LLM Applications 2025.

## Tech Stack

- **Frontend:** React 18+ / TypeScript / Tailwind CSS / shadcn/ui / Vite
- **Data:** Static OWASP Top 10 LLM dataset compiled at build time
- **Infrastructure:** AWS S3 / Cloudflare DNS + SSL / Terraform
